
# hex转中文(utf-8,unicode)

————————————————————————————————————————————

## 本脚本用于把16进制字符串转换成unicode编码的中文字符

脚本语言：python3

脚本文件：2023_11_07_defin.py

使用方法：1、新建hex.txt文件，将需要转码的16进制字符串复制进去，保存退出

2、输入命令      
       python3 2023_11_07_defin.py
                

————————————————————————————————————————————

### 提醒：本脚本支持16进制字符串格式为
0xe5 0x88 0x96
         
0xe5,0x88,0x96
         
%e5 %88 %96
         
%e5,%88,%96
         
e5 88 96
         
e5,88,96
         
e58896

本脚本支持多行输入，单行排序输入
